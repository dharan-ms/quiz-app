# QuizForge API

Express + Prisma backend for QuizForge.

## Scripts
- `npm run dev`
- `npm run start`
- `npm run prisma:generate`
- `npm run db:push`
- `npm run db:seed`
- `npm test`

## API Docs
- Swagger UI: `/api-docs`
- OpenAPI file: `docs/openapi.yaml`

## Seed Credentials
- Admin: `admin@quizapp.com` / `Admin@12345`
- User: `user@quizapp.com` / `User@12345`
